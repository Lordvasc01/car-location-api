## Car Location API 
A set of APIs for exploring car locations data

## Filter available:
- limit
	- N-number
- order
	- by id
	- by location 
	- by trip status
- skip (pagination)
	- N-number
	- per_page
- where 
	- id
	- location 
	- trip status


## Special features:
- Search car within n-km that currently available
