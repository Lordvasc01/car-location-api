## Car Location API 
A set of APIs for exploring car locations data

## Filter available:
- order
	- by id
	- by location 
	- by trip status
- pagination
	- from
	- size
- where 
	- id
	- location_name
	- trip status


## Special features: SEARCH BY COORDINATES
- Search available car (is_on_trip == true) with different radius from certain point
