## Car Location API 
A set of APIs for exploring car locations data

## Filter available:
- order
	- by id
	- by location_name 
	- by trip status
- pagination
	- from
	- size
- where 
	- id
	- location_name
	- trip status


## Special features: SEARCH BY COORDINATES
- Search available car within n-km from certain coordinates

## How to run locally:
- Traditional
- Containerization
